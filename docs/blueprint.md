# **App Name**: HealthReach

## Core Features:

- Hero Section with Registration: A landing page hero section displaying title, description, and a 'Register Now' button that links to the registration form.
- Upcoming Camps Display: Displays a section showcasing upcoming medical camps with details like date, location, and a registration link for each camp, formatted as cards.
- User Registration Form: A comprehensive registration form to capture user details, including Name, Age, Gender, Phone, Email, Address, Camp Location, and Medical Condition, with input validation.
- Admin Dashboard with User Management: An admin interface with a table listing registered users, complete with filtering and search functionalities. Each row includes user's Name, Age, Gender, Phone, Email, Camp Location, and Condition. Also includes Delete and View Details options for each record.
- Informational Pages: Dedicated About and Contact pages to provide details on the organization's mission, team, background, and contact information, along with a functional contact form.

## Style Guidelines:

- Primary color: Soft blue (#74CAFF), evoking a sense of calmness and reliability.
- Background color: Light gray (#F5F7FA), providing a clean and modern feel.
- Accent color: Light green (#A2E79E), representing health and growth.
- Headline font: 'Poppins', a geometric sans-serif for a modern and readable headline.
- Body font: 'PT Sans', a humanist sans-serif ensuring comfortable readability for detailed text.
- Use Font Awesome or similar library with consistent, line-style icons, in blue and green tones.
- Minimalist layout with plenty of white space, using rounded cards for content sections to provide a friendly feel.
- Subtle hover animations on buttons and cards, with smooth transitions to enhance user experience without being distracting.
